# Audio Files

Place your audio files here:

- `ambient.mp3` - Ambient background music for the experience
- `click.mp3` - Sound effect when clicking on stars

## Note

The application will work without these files (using fallback audio generation), but for the best experience, add:
- A soft, ambient cosmic soundtrack (loopable)
- A gentle chime or sparkle sound for star clicks

You can find royalty-free ambient music and sound effects at:
- https://freesound.org
- https://pixabay.com/music
- https://freemusicarchive.org
